Azure Cross-Platform Storage Explorer

A cross-platform implementation of azure storage explorer allowing the manipulation of azure blob storage accounts and eventually tables and queues.

Development

Prerequisites

You will need the following things properly installed on your computer.

  • Git
  • Node.js (with NPM)
  • Bower
  • Ember CLI
  • PhantomJS

Running / Development

  • ember nw

Running Tests

All tests suites can be run with npm test.

Functional and unit tests can be run with ember test and ember test --server. Code Style tests with JSHint and JSCS can be run with grunt test.
